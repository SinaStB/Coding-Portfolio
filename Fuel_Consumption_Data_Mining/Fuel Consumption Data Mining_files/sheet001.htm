* Variable names:
column A    -  id      Identification number for model
column B    -  origin  Origin of the car
column C    -  maker   Auto maker
column D    -  model   Model name (abbreviation)
column E    -  yr      Year
column F    -  cyl     Number of cylinders
column G    -  2wh     2=Two wheel drive, 4=Four wheel drive
column H    -  auto    1=Automatic transmission, 0=otherwise
column I    -  p/s     1=Power steering, 0=otherwise
column J    -  a/c     O=Air conditioning, X=otherwise
column K    -  fro     1=Front wheel drive, 0=otherwise
column L    -  wght    Weight (pounds)
column M    -  disp    Displacement (cubic inches)
column N    -  hp      Horsepower
column O    -  lngth   Length (inches)
column P    -  wdth    Width (inches)
column Q    -  wb      Wheel base (inches)
column R    -  reli    Reliability index (from Consumer Reports)
column S    -  fid     Firm identification number
column T    -  dom     yes=U.S. built, no=imported
column U    -  eur     Y=European model, N=otherwise
column V    -  sales   Sales
column W    -  price   List price (dollars)
column X    -  markup  Estimated markup (thousands of dollars)
column Y    -  mpg     Miles per gallon
